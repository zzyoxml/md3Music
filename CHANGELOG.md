# MD3Music v2.2.0 更新日志

## 🎉 新功能

- **播放器「添加到歌单」** - 播放器页面右上角三点菜单新增「添加到歌单」入口，点击后弹出用户创建的歌单列表，可将当前播放歌曲添加至所选歌单

## 🔧 修复

- **修复荣耀/vivo 手机播放时出现通知提示音** - 优化通知渠道配置，设置静音并禁用震动，兼容各品牌机型
- **修复荣耀平板 V8 Pro 播放时音量忽高忽低** - 移除音频焦点 duck 事件的音量调整逻辑，避免频繁 duck/unduck 导致音量波动
- **修复歌单列表显示不全** - 「添加到歌单」弹窗只显示用户自己创建的歌单，过滤掉收藏的歌单和专辑
- **修复添加到歌单失败** - 修正 API 调用参数格式（歌名|hash|albumId|albumAudioId）
- **修复「我创建的歌单」无批量删除按钮** - 歌单分区头部新增「管理」按钮，支持批量选择并删除歌单

## ⚡ 优化

- **优化添加到歌单/收藏的同步速度** - 改为乐观更新 + 后台同步，点击后瞬间响应，不再阻塞等待服务器返回
- 搜索功能添加缓存机制，优化页面切换体验
- 发现页添加数据新鲜度缓存，减少重复请求
