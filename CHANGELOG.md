# 更新日志

所有重要的项目变更都将记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-17

### ✨ 首次发布

这是 MD3Music 的第一个正式发布版本！

### 🎵 在线音乐功能
- 在线音乐搜索（歌曲、专辑、歌单）
- 每日推荐歌曲
- 热门排行榜展示
- 私人 FM 猜你喜欢
- 主题歌单和场景音乐
- 歌手详情页

### 🎧 播放器功能
- 多音质选择（128k/320k/FLAC）
- 播放速度调节（0.25x - 4x）
- 循环模式（单曲/列表/随机）
- 歌词同步显示
- 播放列表管理
- 后台播放通知（Android）
- 预加载下一首歌曲

### 📱 用户中心
- VIP 签到领取
- 收藏歌曲管理
- 播放历史记录
- 下载管理
- 离线播放支持

### 🎨 设计与体验
- Material Design 3 设计规范
- 动态颜色主题
- 响应式布局（手机/平板/桌面）
- 深色模式支持
- 浅色/深色/跟随系统主题切换

### 🔧 技术实现
- Provider 状态管理架构
- just_audio 音频播放引擎
- Dio 网络请求库
- SQLite 本地数据存储
- 酷狗音乐 API 集成
- Web 平台兼容性适配

### 📦 平台支持
- Android（主要支持平台）
- Web（部分功能受限）
