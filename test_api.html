关于API服务器

当前API服务器地址: http://localhost:3000

如果需要修改API服务器地址，请在Flutter应用的"设置"页面中修改。

Android版本的注意事项

⚠️ Android设备上的"localhost"指向设备本身，不是电脑。

✅ 解决方案：

  1. 确保API服务器在电脑上运行（端口3000）
  2. 获取电脑的局域网IP地址（例如：192.168.1.100）
  3. 在Android应用的"设置"中，将API服务器地址改为：http://192.168.1.100:3000
  4. 确保Android手机和电脑在同一WiFi网络下

查找电脑IP地址

Windows: 打开CMD，输入 ipconfig，找到"IPv4 地址"

或者运行: Get-NetIPAddress -AddressFamily IPv4 | Where-Object { $_.InterfaceAlias -like "*Wi-Fi*" -or $_.InterfaceAlias -like "*Ethernet*" } | Select-Object IPAddress
