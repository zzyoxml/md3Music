# 修复收藏/红心功能 + 实时刷新

## 问题分析

### 问题1: 红心取消后"我喜欢"页面不刷新
**根因**: `favorites_page.dart:43` 的 `_onFavoritesChanged` 检查 `_selectedPlaylist!.id == '2'`，但"我喜欢"歌单的实际 `id` 是 `collection_3_884062510_2_0`（globalCollectionId），不是 `'2'`。所以条件永远为 false，`_loadPlaylistSongs` 从未被调用。

### 问题2: 收藏歌单报错 30203
**根因**: `PlaylistPage._collectPlaylist()` 调用 `api.createPlaylist()`，传入 `listCreateUserid` 和 `listCreateListid`。但 `KugouPlaylistBrief.toPlaylist()` 中这些字段可能为 null，导致 `list_create_userid=0` 和 `list_create_listid=0`。服务端 `playlist_add.js` 用这些值请求酷狗API，返回 30203（参数无效）。

### 问题3: 无法取消收藏歌单
**根因**: `PlaylistPage` 只有 `_collectPlaylist()` 方法，没有取消收藏的功能。`_isCollected` 始终初始化为 `false`，无法检测已收藏状态，也没有取消收藏按钮。

## 修改方案

### 1. 修复"我喜欢"实时刷新
**文件**: `lib/modules/user/favorites_page.dart:41-48`

将 `_onFavoritesChanged` 的判断改为匹配 `globalCollectionId` 中包含 `_2_` 的模式（"我喜欢"歌单的 listid 固定为 2），或者直接不判断——只要在"我喜欢"页面就刷新。

```dart
void _onFavoritesChanged() {
  if (_selectedPlaylist != null && mounted) {
    // "我喜欢"歌单固定 listid=2，检查 globalCollectionId 或 id 是否包含 _2_
    final gid = _selectedPlaylist!.globalCollectionId ?? _selectedPlaylist!.id;
    final isMyFavorite = gid.contains('_2_') || _selectedPlaylist!.name == '我喜欢';
    if (isMyFavorite) {
      _loadPlaylistSongs(_selectedPlaylist!, noCache: true);
    }
  }
}
```

### 2. 修复收藏歌单参数
**文件**: `lib/modules/playlist/playlist_page.dart:34-62`

`_collectPlaylist()` 中，当 `listCreateUserid`/`listCreateListid` 为 null 时，应该从 `globalCollectionId` 解析，而不是传 null（服务端会转为 0）。

```dart
String? listCreateUserid = widget.playlist.listCreateUserid;
String? listCreateListid = widget.playlist.listCreateListid;
if ((listCreateUserid == null || listCreateListid == null) && widget.playlist.id.contains('_')) {
  final parts = widget.playlist.id.split('_');
  if (parts.length >= 4) {
    listCreateUserid ??= parts[2];
    listCreateListid ??= parts[3];
  }
}
```

### 3. 添加取消收藏歌单功能
**文件**: `lib/modules/playlist/playlist_page.dart`

- 在 `initState` 中检查是否已收藏（通过 `getUserPlaylist` 对比 listid）
- 添加 `_uncollectPlaylist()` 方法调用 `api.deletePlaylist(listid)`
- 在收藏按钮旁显示取消收藏按钮（当 `_isCollected == true` 时）

### 4. 收藏/取消收藏后刷新收藏列表
**文件**: `lib/modules/user/favorites_page.dart`

在 `FavoritesPage` 的 `_buildPlaylistList` 中，当从 `PlaylistPage` 返回时调用 `_loadPlaylists()` 刷新歌单列表。通过 `Navigator.push` 的 `then` 回调实现。
